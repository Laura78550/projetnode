# Projetnodejs-music

# Dependencies
- nodeJs 12.13.0 et +
- npm ou yarn ou pnpm

# API documentation
An API is available to get the lyrics of a song. The documentation is available on Apiary.io: http://docs.lyricsovh.apiary.io/.
